Project Overview

Home Defense Camera is created to be an affordable security system containing a face-detecting camera which can be monitored in Home Assistant

Benefit

  • Detecting Intruders
  • Alert Light Bulb
  • Access by Home Assistant

Block Diagram

Hardware Components

  • Raspberry Pi 4B + Powersupply



  • ESP32 Boards



  • Webcam



  • Servo Motor + Bracket



  • Zigbee Light Bulbs

Software Components

  • Home Assistant OS
  • Visual Studio Code
  • ESPHome
  • OpenCV
  • MQTT
  • Zigbee2mqtt

Implementation

1) Set up Home Assistant and a local MQTT Broker with Raspberry Pi 4

2) Install Related Add ons including ESPHome, MJPG IP Camera and MQTT

3) Run a python scripts with desired mode, MQTT or Serial. To a connected webcam

4) For MQTT mode, compile and upload ESPHome YAML Config to ESP32

5) For Serial mode, upload arduino sketch to ESP32
