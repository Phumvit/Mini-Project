Project Overview

Home Defense Camera is created to be an affordable security system containing a face-detecting camera, turret and rfid reader which can be monitored and controlled in Home Assistant

Benefit

  • Detecting Intruders
  • Authenticating user using RFID
  • Surveillance Turret

Hardware Components

  • Raspberry Pi 4B + Powersupply

  • ESP32 Boards
  • Webcam
  • Servo Motor + Bracket
  • RFID Reader and Tags
  • Zigbee Light Bulbs

Software Components

  • Home Assistant OS
  • Visual Studio Code
  • ESPHome
  • OpenCV
  • MQTT
  • Zigbee2mqtt
